## How to run the project
1. Install [node.js](https://nodejs.org/en/download)
2. Install [pnpm](https://pnpm.io/installation)
3. In the cloned repository, install dependencies
   ```bash
   cd git-together-app
   pnpm i next@latest react@latest react-dom@latest
   ```
4. In the folder git-together-app, create a file .env.local. Email hschul7@lsu.edu to get the contents of the file to paste into .env.local.
5. Run the development server
   ```bash
   pnpm dev 
   ```
6. Navigate to localhost:3000/login to start testing
